<h2>🧐 Features</h2>

Here're some of the project's best features:

*   ⚡ Instant password generation using random.SystemRandom
*   📏 Adjustable password length via slider (6 – 64 characters)
*   🔠 Toggle character sets: Uppercase Lowercase Numbers Symbols
*   ⚠️ Option to avoid ambiguous characters (O 0 l 1 I)
*   📊 Real-time strength meter (Weak / Medium / Strong)
*   📋 One-click copy to clipboard
*   🗑 Clear button to reset
*   🎨 Premium dark UI with smooth animations

<h2>🛠️ Installation Steps:</h2>

<p>1. Clone the repository:</p>

```
git clone https://github.com/xcyberspy/Password-Generator.git
```

<p>2. Navigate to the Directory</p>

```
cd Password-Generator
```

<p>3. Install Required Packages</p>

```
pip install customtkinter pyperclip
```

<p>4. Run the application</p>

```
python app.py
```
